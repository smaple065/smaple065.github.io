COMPUTER SCIENCE

Falcon Fitness | 2019

This web application is home to descriptions and tutorials of exercises that can be done in fitness centers around the campus of Seattle Pacific University. For more information check out this poster.
Built using ReactJS | Hosted on Google Firebase

Please Don't Touch the Pellets | 2019

In this cute-but-chaotic game, you are rolling around a maze trying to collect 10 tokens so that an exit will appear, but watch out-- terrible things happen when you touch the pellets.
Built in Unity3D

Network Collapse | 2018

This is one of three educational games developed at Pacific Nortwest National Laboratory. In this game, the user plays the role of a network component and attempts to route packages to their correct destination. For more information check out this poster.
Built in Unity3D | Visual Design in Google Blocks

ABOUT ME

Hi, I’m Sierra! I am a recent graduate from Seattle Pacific University with a Bachelor’s Degree in Computer Science and a minor in Art – 2D Media, which essentially means that I write creative code and make meticulously calculated art.

I enjoy...
